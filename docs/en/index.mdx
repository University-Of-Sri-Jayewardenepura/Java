# Object-Oriented Programming with Java

This module covers Java programming from the ground up, progressing through core language features to advanced object-oriented concepts.

## Topics

| # | Topic | Description |
|---|-------|-------------|
| 1 | [Introduction](./introduction) | Java evolution, features, JVM/JRE/JDK, basic syntax |
| 2 | [Data Types & Operators](./data-types) | Primitives, non-primitives, type casting, operators |
| 3 | [Control Statements](./control-statements) | Selection, iteration, jump statements, methods |
| 4 | [OOP Introduction](./oop-intro) | Software crisis, OOP advantages, four pillars |
| 5 | [Classes & Objects](./classes-objects) | Constructors, instance methods, `this` keyword |
| 6 | [Encapsulation](./encapsulation) | Access modifiers, getters/setters, static members |
| 7 | [Arrays](./arrays) | 1D and 2D arrays, common operations |
| 8 | [Strings](./strings) | String methods, immutability, StringBuilder |
| 9 | [Inheritance](./inheritance) | Types of inheritance, `super`, method overriding |
| 10 | [Polymorphism](./polymorphism) | Compile-time and runtime polymorphism |
| 11 | [Abstraction](./abstraction) | Abstract classes, interfaces |
| 12 | [Exceptions & Threading](./exceptions-threads) | Exception handling, thread lifecycle, synchronization |

## Prerequisites

- A Java Development Kit (JDK) installed (version 8 or later).
- A text editor or IDE (e.g., IntelliJ IDEA, VS Code, Eclipse).

## How to Run Examples

Each topic has a corresponding example in the [`examples/`](https://github.com/University-Of-Sri-Jayewardenepura/Java/tree/main/examples) directory. Compile and run any example with:

```bash
javac ExampleName.java
java ExampleName
```
